2.2 AdSlot

Name

Mandatory

Type

Description

slotId

M

string

展示位Id，通过AGC管理台创建展示位时产生

adCount

M

integer

请求广告数量,榜单类应用列表广告取值不超过30；其它场景广告位只能填1

pageNumber

O

integer

当前请求的页码数,取值1到10，不填写或填写错误可能导致榜单类应用列表广告重复

lastItemIds

O

list<string>

最近曝光过的广告，DSP后台会过滤掉这些广告，不填写或填写错误可能导致短时间内多次请求的广告重复。

installedApp

M

integer

返回应用的安装过滤标识,仅推荐场景使用，取值：1-仅未安装应用 2-仅已安装应用 3-不限；仅限华为内媒生效，外媒获取不到udid；

personalize

O

integer

个性化标识，0-非个性化推荐（所有终端用户看到的广告一样，默认），1-个性化推荐

keywords

O

string

搜索关键词，搜索场景必填；

downloadType

M

integer

媒体侧的应用下载方式：1：通过AGDSDK下载；2：通过deeplink下载（安装方式有限）；

installType

O

integer

安装方式，不同的安装方式有不同的安装体验，downloadType=2时必填，取值范围：1：Mini详情页，手动安装；6：全屏详情页，手动安装；7：Mini详情页，自动安装。自动安装时会有通知栏显示进度条；8：全屏详情页，自动安装，自动安装时会有通知栏显示进度条
